# speed
CLI/TUI утилита для скоростного чтения в терминале. Написана на Rust для максимальной производительности и минимального потребления ресурсов.
## О проекте
Обычные приложения для чтения отвлекают визуальным мусором, а скрипты без интерфейса неудобны в управлении. speed решает эту проблему, предоставляя минималистичный TUI прямо в консоли.
## Возможности
Текущие:
 * Нативный TUI-интерфейс (на базе ratatui).
 * Быстрый парсинг аргументов командной строки (clap).
 * Чтение стандартных текстовых форматов.
 * Настройка скорости чтения (WPM) через аргументы при запуске.
 * Отсутствие зависимостей от тяжелых графических библиотек.
Roadmap:
 * Полноценная поддержка формата EPUB (парсинг глав и метаданных).
 * Динамическая настройка скорости чтения (WPM) горячими клавишами без перезапуска.
 * Навигация по тексту: пауза, перемотка назад/вперед, переход по главам.
## Установка
Для сборки потребуется установленный cargo.git clone https://github.com/CookieCompiler/speed.git
cd speed
cargo build --release

Исполняемый файл будет скомпилирован в директорию target/release/speed.
## Использование
Передайте путь к файлу в качестве аргумента (поддержка формата .epub находится в активной разработке):speed path/to/book.txt

Или:speed path/to/book.epub

### Управление
 * Space — пауза
 * q — выход
 * Стрелки — навигация
